# Mobile PWA Skill

## When To Trigger

Use this when changing layout, navigation, upload flow, or PWA assets.

## Input Format

Describe the page or flow being changed and the target phone viewport.

## Output Format

List the mobile behavior, PWA behavior, and verification steps.

## Rules

- Mobile first.
- Upload flow must stay at 4 steps or fewer.
- Use large buttons and clear touch targets.
- Keep bottom navigation.
- Show a clear image preview.
- Keep forms simple.
- Preserve installable PWA basics.

## Forbidden

- Do not design desktop-only flows.
- Do not hide primary actions in complex menus.
- Do not add heavy animation for MVP.
