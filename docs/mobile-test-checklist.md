本清单用于第六阶段上线前验收。当前开发环境无法直接连接真实手机时，先使用浏览器手机视口模拟测试；真实上线前仍需在至少一台 iPhone 和一台 Android 手机上复测。

## iPhone Safari

- [ ] 可以打开首页、登录页、拍题页、复习页、错题库、报告页、设置页和冲刺页。
- [ ] 登录后刷新页面仍保持登录状态。
- [ ] 未登录访问 `/upload`、`/review`、`/questions`、`/reports`、`/settings`、`/sprint` 会跳转登录页。
- [ ] 页面没有横向滚动。
- [ ] 底部导航不遮挡主要按钮和表单。
- [ ] 表单输入时键盘不遮挡当前输入框和提交按钮。
- [ ] 系统深色模式下文字仍可读，页面不出现白底白字或黑底黑字。

## Android Chrome

- [ ] 可以打开首页、登录页、拍题页、复习页、错题库、报告页、设置页和冲刺页。
- [ ] 登录后刷新页面仍保持登录状态。
- [ ] 未登录访问保护页会跳转登录页。
- [ ] 页面没有横向滚动。
- [ ] 底部导航不遮挡内容。
- [ ] 表单输入时键盘不遮挡提交按钮。
- [ ] 系统主题变化后页面仍可读。

## 拍照和相册上传

- [ ] 手机拍照上传可以唤起后置摄像头。
- [ ] 从相册选择 jpg/jpeg/png/webp 可以生成预览。
- [ ] 非图片格式会显示错误提示，不会进入保存。
- [ ] 大于 8MB 的图片会显示大小限制提示。
- [ ] 大图会尝试压缩，并显示原图大小、上传大小和压缩状态。
- [ ] 压缩失败时可以回退原图上传，不会白屏。
- [ ] 弱网或上传失败后按钮会恢复可点击状态，并显示失败原因。
- [ ] 上传成功后可以进入错题详情，原始 `image_path` 保留。

## 复习和错题详情

- [ ] 今日复习按钮点击后只写入一次复习结果。
- [ ] 复习写入失败会显示错误，不会卡在 loading。
- [ ] 错题详情原题图片可放大查看。
- [ ] 图片 signed URL 失效或生成失败时，页面显示“暂不可预览”，不崩溃。
- [ ] AI 分析失败时显示错误提示，不崩溃。
- [ ] `verified` 题干重新分析默认保留人工核对题干。
- [ ] 错题编辑保存失败时显示错误，不破坏 `image_path`。

## 报告和导出

- [ ] 报告为空时显示空状态说明，不裸显 JSON。
- [ ] 报告页在 390px 宽度下文字可读。
- [ ] `/settings` 可导出 JSON。
- [ ] `/settings` 可导出 Markdown。
- [ ] `/settings` 可导出 CSV。
- [ ] 导出文件名包含当天日期，例如 `11408-review-export-2026-06-04.json`。

## PWA

- [ ] iPhone Safari 可以通过分享菜单添加到主屏幕。
- [ ] Android Chrome 可以安装或添加到桌面。
- [ ] 桌面入口打开后仍可登录和访问核心页面。
- [ ] manifest 名称、短名称、主题色和图标显示正常。

## 冲刺和批量管理

- [ ] `/sprint` 可以显示高危错题。
- [ ] 冲刺卡片显示缩略图、危险原因和错题详情入口。
- [ ] 冲刺页可以按科目筛选。
- [ ] 冲刺页“标记已掌握”成功后卡片移出当前列表。
- [ ] `/questions` 可以多选错题。
- [ ] 批量标记已掌握前会确认。
- [ ] 批量标记 `needs_fix` 前会确认。
- [ ] 批量加入冲刺复习前会确认。
- [ ] 批量操作成功有提示，失败有错误提示。

## 第六阶段已修复或重点防护项

- 页面级 CSS 禁止横向滚动，图片和长文本默认限制宽度并断行。
- 上传、复习、编辑和分析操作都有失败后恢复 loading 的处理。
- Supabase 环境变量缺失时页面显示配置提示，不白屏。
- 保护路由缺少登录时跳转 `/login?next=...`。
- 日期计算优先使用用户 timezone，缺失时默认 `Asia/Shanghai`。
- 薄弱点统计中的中文掌握状态常量已修正，避免统计不命中。
